ScalaTest Plus

The ScalaTest Plus project aims to define integration libraries between ScalaTest and other libraries, to make it easier to test code using those libraries. The first such integration library is ScalaTest + Play.

Each integration library will depend on a version of ScalaTest as well as a version of the target library. The package name of each ScalaTest Plus library will start with org.scalatestplus. For example, the package name for ScalaTest + Play is org.scalatestplus.play.
