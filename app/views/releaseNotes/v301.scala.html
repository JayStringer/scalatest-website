ScalaTest/Scalactic 3.0.1 Release Notes

ScalaTest/Scalactic 3.0.1 includes the enhancements and bug fixes listed below. No source code using ScalaTest/ScalaUtils 3.0 or higher should break, but from any version other than other 3.0.x releases (with which this release is binary compatible), you will likely need to do a clean build to upgrade.

For information on how to include ScalaTest in your project, see the download page. For information on how to use Scalactic in your production code, see its download page.

Enhancements

  • Eliminated unnecessary casts in Or.
  • Unregister reporter to free up memory when completedTests is called in SuiteSortingReporter
  • Changed PositionMacro to only put in file path name when environment variable SCALACTIC_FILE_PATHNAMES is set to true, the default off behavior enable reproducable test build.
  • Updated the ScalaCheck dependency to version 1.13.4.
  • Updated the Scala-js dependency to version 0.6.13.

Bug fixes

  • Fixed typecheck bug in ConfigMap.
  • Fixed output of nested suites not being displayed in SBT problem.
  • Fixed timer not canceled problem in TimeLimitedTests. (reported as github issue #981))
  • Fixed slowpoke alert not being displayed in SBT problem.

Visit ScalaTest Release Notes for links to the release notes of all previous versions, or step back in time by visiting the release notes for the previous version.
